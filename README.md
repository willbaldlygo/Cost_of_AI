# The True Cost of AI

An interactive, educational, single-page website showing the electricity, water, and carbon cost of different AI tasks — with relatable everyday comparisons and practical, research-backed advice on reducing your AI environmental footprint.

## Tech Stack

- **HTML / CSS / Vanilla JavaScript** — no frameworks, no build tools, no npm
- Static site designed for GitHub Pages

## Running Locally

Just open `index.html` in your browser. No server required.

## Data Sources

All figures are compiled from 32 peer-reviewed and institutional sources. See the "Research Sources" panel on the site for the full reference list.
